# Icon Generation Instructions

The placeholder icon files have been created but need to be replaced with actual images.

## Required Icons:
1. **icon-192.png** - 192x192px PWA icon
2. **icon-512.png** - 512x512px PWA icon
3. **favicon-32.png** - 32x32px favicon
4. **favicon-16.png** - 16x16px favicon
5. **apple-touch-icon.png** - 180x180px iOS icon

## How to Generate:
1. Open `icon-placeholder.html` in a browser
2. Right-click each canvas and save as the corresponding filename
3. Replace the empty placeholder files with the saved images

## Alternative:
Use the `icon.svg` file to generate PNG versions using an image editor or online converter.

## Icon Design:
- Dark gray background (#1f2937)
- Red mine symbol (#dc2626)
- Black spikes
- White highlight for depth
